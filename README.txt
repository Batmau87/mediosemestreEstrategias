# Prototipo HTML · USFQ Medios Interactivos

Este paquete está listo para subirse a GitHub Pages.

## Qué hacer
1. Sube el contenido del zip a un repositorio.
2. Deja `index.html` en la raíz.
3. En GitHub, ve a **Settings > Pages**.
4. En **Build and deployment**, elige **Deploy from a branch**.
5. Selecciona la rama principal y la carpeta raíz.
6. Guarda y espera a que GitHub publique el sitio.

## Archivo principal
- `index.html`

No necesita instalación, npm ni build.
